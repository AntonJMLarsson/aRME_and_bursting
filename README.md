Repository for transcriptional burst analysis from allelic single-cell RNA-seq described in the article **Transcriptional bursts explain autosomal random monoallelic expression and affect allelic imbalance** by Anton J.M. Larsson, Christoph Ziegenhain, Michael Hagemann-Jensen, Bj√∂rn Reinius, Tina Jacob, Tim Dalessandri, Gert-Jan Hendriks, Maria Kasper and Rickard Sandberg. PLOS Computational Biology. doi: DOI HERE.

## Folders
/data contains all the data used to produce the figures
/figures contain the produced figures
/inference contain the transcriptional burst inference code. Find more at code [txburst](https://github.com/sandberg-lab/txburst).
The root folder contain notebooks to reproduce the figures and analysis, and a STAN model used for figure 5.
